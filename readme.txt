Instructions for Clemson School of Computing Linux lab machines:

1. Download and unzip the project into your home directory (e.g., `~/ClemsonQuest`).
2. Open a terminal and `cd` into the unzipped folder: `cd ~/ClemsonQuest`.
3. Load Node/Expo prerequisites if needed (example: `module load node/20.11.1`).
4. Install dependencies once with `npm install`.
5. Run `npx expo start --tunnel` to launch the Expo dev server (tunnel works best on lab firewalls).
6. Use the QR code / web UI in the terminal to open the app on a device or browser.
